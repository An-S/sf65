- sf65 examples -

These examples demonstrate the capabilities of sf65. A simple source served as a base. It displays a screen with magenta on the left and cyan on the right. The source code has been distorted a little bit to emphasize the effect of the source formatter.

Setting output column for mnemonics

This example shows how the -m flag affects the output of the mnemonics. It is notes that the -m flag does not change the output column of the operands. This is specified by the -o flag. The numerical argument after -m was varied from 8 to 16 in steps of two. The sources have been truncated at column 20 to be able to display the formatted sources side by side for the different parameters.
